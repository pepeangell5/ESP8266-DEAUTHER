Web Flasher para ESP8266-DEAUTHER

Instala el firmware PEPEANGELL directamente desde Chrome o Edge usando USB. La pagina carga el binario publicado en este repositorio y lo envia al ESP8266 mediante Web Serial.

Descargar binario Ver repositorio

Instalador

Usa Chrome o Edge con soporte Web Serial. Abre esta pagina por HTTPS para habilitar Web Serial.

Si tu placa no entra sola en modo bootloader, manten presionado FLASH/BOOT mientras conectas o reinicias el ESP8266.

Version2.0.0
ChipESP8266 / NodeMCU
AP despues del flashPepeAngell-DEAUTH
IP Dashboard192.168.4.2

Flujo de instalacion

1

Conecta el ESP8266

Usa un cable USB de datos y selecciona el puerto serial cuando el navegador lo pida.

2

Inicia el flasheo

Presiona conectar e instalar. El navegador cargara el manifest y el binario desde la carpeta binarios.

3

Espera el reinicio

Cuando termine, deja que el ESP8266 reinicie y cree su red de administracion.

4

Entra al dashboard

Conectate al AP y abre `http://192.168.4.2` para usar la interfaz web.

Este firmware es para investigacion, aprendizaje y pruebas autorizadas en redes propias o laboratorios controlados.
